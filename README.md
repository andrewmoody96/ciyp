## Changed in 2026.04 update:

- **Migrated from Remix v2 to [React Router v7](https://reactrouter.com/):**
  - Remix has been merged into React Router by the same team
  - Updated Vite plugin from [`@remix-run/dev`](https://www.npmjs.com/package/@remix-run/dev) to [`@react-router/dev`](https://reactrouter.com/dev/framework/start/installation)
  - Updated server rendering with [`@react-router/express`](https://reactrouter.com/explanation/special-files#entry-server) and [`@react-router/node`](https://www.npmjs.com/package/@react-router/node)
  - Added [file-based routing](https://reactrouter.com/how-to/file-route-conventions) via `@react-router/fs-routes`

- **Completed [TypeScript](https://www.typescriptlang.org/) migration:**
  - Converted all `.jsx` components to `.tsx`
  - Replaced runtime `prop-types` with TypeScript interfaces
  - Added type annotations to utility functions
  - Removed `prop-types` and `@types/prop-types` dependencies

- **Dependency updates:**
  - [`googleapis`](https://github.com/googleapis/google-api-nodejs-client) 148 -> 171
  - [`cross-env`](https://www.npmjs.com/package/cross-env) 7 -> 10
  - [`dotenv`](https://www.npmjs.com/package/dotenv) 16 -> 17
  - [`isbot`](https://www.npmjs.com/package/isbot) 4 -> 5
  - [`typescript`](https://www.typescriptlang.org/) 5 -> 6

- **Cleanup:**
  - Removed unused `@fortawesome/fontawesome-free` package
  - Removed broken Font Awesome CSS & font references
  - Simplified Google Calendar API authentication for googleapis v171
